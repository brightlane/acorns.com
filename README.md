---

## What This Is

AcornsGuide is an affiliate marketing site targeting people considering the Acorns micro-investing app. It covers reviews, competitor comparisons, fee analysis, and how-to guides — linking readers to Acorns via an affiliate partner link.

The entire site is generated by a single Python script (`build.py`) that outputs static HTML. It deploys automatically to GitHub Pages via GitHub Actions.

---

## How It Works

```
build.py  →  docs/  →  GitHub Pages  →  live site
```

1. `build.py` runs and writes 19 HTML pages + support files into `docs/`
2. GitHub Actions commits `docs/` back to the repo and deploys to GitHub Pages
3. The site rebuilds automatically every day at 6 AM UTC

## Adding or Editing Content

All content lives inside `build.py` in the `PAGES` list. Each page is a Python dict:

```python
PAGES.append(dict(
    slug="page-slug",
    title="Page Title | AcornsGuide",
    desc="Meta description for Google.",
    kw="keyword one, keyword two",
    h1="H1 Heading — <span>Highlighted Part</span>",
    tag="Eyebrow Label · Subtitle",
    hero_cta="🌱 Button Text",
    crumbs=[("Home","index.html"),("Page Name",None)],
    faqs=[
        ("Question?", "Answer."),
    ],
    body="""
    <section class="cb">
      <p class="lead">First paragraph...</p>
      <h2>Section heading</h2>
      <p>Body content. Use {A} for the affiliate link.</p>
      <a href="{A}" class="cta" target="_blank" rel="noopener sponsored">Button</a>
    </section>
    """
))
```

Use `{A}` anywhere in the body to insert the affiliate link with UTM tracking.

## Running Locally

```bash
python build.py
```

Creates a `docs/` folder with all HTML files. Open `docs/index.html` in a browser to preview.

**Requirements:** Python 3.7+ — no packages needed, standard library only.

---

## Deploying

Push any change to `main` — GitHub Actions builds and deploys automatically.

To trigger manually: **Actions → Build & Deploy AcornsGuide → Run workflow**
